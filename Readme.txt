🛠 التقنيات المستخدمة

اللغات الأساسية

· HTML5 - بناء الهيكل الأساسي مع استخدام العناصر الدلالية
· CSS3 - التصميم المتقدم باستخدام CSS Grid و Flexbox
· JavaScript (ES6+) - إضافة التفاعلية والوظائف الديناميكية

أطر العمل والمكتبات

· Bootstrap 5 - الإطار الأمامي للتصميم المتجاوب 
· AOS (Animate On Scroll) - إضافة حركات عند التمرير
· Swiper.js - للعروض التقديمية والشرائح المنزلقة
· Isotope.js - نظام تصفية وعرض المشاريع
· GLightbox - معرض الصور المدمج
· Boxicons - مجموعة أيقونات شاملة

أدوات التطوير والنشر

· Git & GitHub - التحكم بالإصدارات
· Vercel - منصة النشر والاستضافة
· Google PageSpeed Insights - تحسين الأداء

🚀 كيفية تشغيل المشروع محلياً

المتطلبات الأساسية

· متصفح ويب حديث (Chrome 90+, Firefox 88+, Safari 14+)
· محرر نصوص (VS Code, Sublime Text, Atom)
· اتصال بالإنترنت (لتحميل المكتبات الخارجية)

خطوات التشغيل المفصلة

1. استنساخ المستودع:
   ```bash
   git clone https://github.com/WWW-Alhnani-COM/portfolio.git
   cd portfolio
   ```
2. تشغيل خادم محلي:
   ```bash
   # الطريقة 1: استخدام Python
   python -m http.server 8000
   
   # الطريقة 2: استخدام Node.js (إذا كان مثبتاً)
   npx http-server
   
   # الطريقة 3: استخدام PHP
   php -S localhost:8000
   ```
3. فتح المتصفح:
   · اذهب إلى http://localhost:8000
   · أو افتح ملف index.html مباشرة في المتصفح

🎯 دليل الاستخدام

الصفحة الرئيسية (Home)

· المقدمة الشخصية: قسم ترحيبي مع صورة شخصية ونبذة مختصرة
· الإحصائيات: عرض إحصائيات سريعة (عدد المشاريع، سنوات الخبرة، العملاء)
· أحدث المشاريع: معرض لأهم 3 مشاريع حديثة

صفحة عني (About)

· السيرة الذاتية: معلومات شخصية ومهنية مفصلة
· الخبرات العملية: قائمة بالوظائف والإنجازات
· المهارات التقنية: تصنيف المهارات حسب المستوى
· الشهادات والتعليم: المؤهلات العلمية والدورات

صفحة المشاريع (Projects)

· نظام التصفية: تصفية المشاريع حسب التقنية (Web, Mobile, Design)
· تفاصيل المشروع: عرض شامل لكل مشروع مع الصور والشرح
· الروابط: روابط مباشرة للمشاريع الحية وروابط GitHub

صفحة الخدمات (Services)

· قائمة الخدمات: التطوير الويب، التطبيقات، UI/UX Design
· أسعار الخدمات: باقات مختلفة مع المميزات
· عملية العمل: خطوات تنفيذ المشاريع

صفحة التواصل (Contact)

· نموذج التواصل: نموذج متكامل مع التحقق من البيانات
· معلومات الاتصال: البريد، الهاتف، وسائل التواصل الاجتماعي
· الموقع: خريطة تفاعلية للموقع

🔧 التخصيص والإعدادات

تغيير المعلومات الشخصية

1. افتح ملف index.html
2. عدل المعلومات في الأقسام التالية:
   ```html
   <!-- تغيير الاسم والوصف -->
   <header>
       <h1>Mohammed AL-Hnani</h1>
       <p>Full-Stack Developer</p>
   </header>
   ```

إضافة مشاريع جديدة

1. أضف صور المشروع في assets/img/projects/
2. عدل ملف assets/js/filter.js:
   ```javascript
   const projects = [
       {
           id: 1,
           title: "اسم المشروع الجديد",
           category: "web",
           image: "assets/img/projects/new-project.jpg",
           link: "#",
           technologies: ["React", "Node.js", "MongoDB"]
       }
   ];
   ```

تغيير الألوان والثيم

1. عدل ملف assets/css/colors.css:
   ```css
   :root {
       --primary-color: #2c3e50;
       --secondary-color: #34495e;
       --accent-color: #e67e22;
       --text-color: #ecf0f1;
   }
   ```

🐛 استكشاف الأخطاء وإصلاحها

مشاكل شائعة وحلولها

المشكلة الحل
الصور لا تظهر تحقق من مسارات الملفات وأحرف الصغيرة والكبيرة
الحركات لا تعمل تأكد من تحميل مكتبة AOS بشكل صحيح
النماذج لا ترسل تحقق من اتصال الإنترنت وإعدادات JavaScript

الإبلاغ عن الأخطاء

1. اذهب إلى صفحة Issues في GitHub
2. انقر على "New Issue"
3. صف المشكلة بالتفصيل مع:
   · المتصفح المستخدم
   · خطوات إعادة Producing المشكلة
   · لقطات شاشة إذا أمكن
